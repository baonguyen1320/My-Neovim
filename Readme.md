# Prerequisites:

1. Install vim-plug
```
    curl -fLo ~/.vim/autoload/plug.vim --create-dirs \
    https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim
```

2. Install ripgrep for Telescope
```
brew install ripgrep
```

*** Some command useful
```
:checkhealth
:checkhealth telescope
```
